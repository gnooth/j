The Status Bar

The status bar at the bottom of each frame displays information about the buffer that is currently active in that frame. By default, this information includes the mode of the buffer (Java, C++, HTML, etc.) and the location of the caret (line and column number). If macro recording is in progress, "REC" is displayed in the status bar (at the right end, after the column number).

In addition to the default information, the status bar can optionally display the line separator for the current buffer (LF, CR, or CR+LF) and the total number of lines in the buffer. To activate these options, add one or both of the following lines to your preferences file: 

    StatusBar.displayLineSeparator=true
    StatusBar.displayLineCount=true

The line separator for the current buffer can be changed in the Properties dialog (see properties).

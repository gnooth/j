Directory Buffers

Directory buffers let you browse for files and perform basic file management functions.

You can open a directory buffer by specifying the name of the directory in the Open File dialog or on the command line when starting j. The command dir, mapped by default to Ctrl D, opens a directory buffer on the current directory.

The command dirHomeDir, available on the tool bar, opens a directory buffer on your home directory.

The following commands are specifically relevant to directory buffers:

  • dir
  • dirCopyFile
  • dirCycleSortBy
  • dirDeleteFiles
  • dirDoShellCommand
  • dirHomeDir
  • dirLimit
  • dirMoveFile
  • dirOpenFile
  • dirOpenFileAndKillDirectory
  • dirRescan
  • dirTagFile
  • dirUnlimit
  • dirUpDir
